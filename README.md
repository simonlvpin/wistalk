# TalktoCEO

这是一个浏览器端单页 Web 程序，用来把 CEO 讲话类 Word 文档拆解成可学习的话题文章。

## 使用方式

1. 打开本地页面。
2. 使用账号登录。默认超级管理员是 `Simon.Lv@fanruan.com`。首次部署后请在本地初始化或修改管理员密码。
3. 上传 `.docx`、`.txt`、`.md` 文件，或直接粘贴讲话正文。
4. 点击“保存并拆解”。
5. 在“总览、话题文章、深挖学习、金句、深化路径”之间切换学习。
6. 使用“复制”或“导出”获得 Markdown 版本。

## 当前能力

- 浏览器内解析 `.docx` 文件，不依赖后端服务。
- 支持本地用户管理：申请试用、管理员生成初始密码、登录、退出、修改密码、找回密码申请。
- 登录时可选择记住登录状态：默认 7 天，也可以选择 30 天或永远。
- 只有 `@fanruan.com` 邮箱可以申请试用。
- 超级管理员 `Simon.Lv@fanruan.com` 可以在管理后台查看用户、注册时间、最近登录、登录次数、访问次数和找回申请。
- 普通用户只能看到自己上传的讲话；超级管理员可以看到所有用户上传的讲话。
- 左侧侧边栏仅提供一级模块：普通用户为学习总览、分类管理；管理员额外拥有后台管理。
- 学习总览包含核心总览、话题文章、深挖学习、金句库、方法论的步骤式学习路径。
- 文档使用战略、组织管理、文化建设、AI、数字化、其他等固定分类，也允许新增自定义分类；管理员可以看到全部分类。
- 管理员可以按资料拥有者和文档分类筛选文章；用户可手动打标签和快速修改文档分类。
- 账号设置提供 10 种个人界面配色选择。
- 自动保存上传或粘贴的讲话到浏览器本地 IndexedDB。
- 浏览器 IndexedDB 中的用户、文档、API Key 和分析数据不会进入 Git 仓库；请勿提交 `.db`、`.sqlite`、`.env` 或浏览器导出的数据文件。
- 在“讲话文档库”里按标题、标签、正文搜索，并快速切换已保存文档。
- 支持给文档打标签，例如“战略、AI、组织变革”。
- 自动按战略、数字化、AI、组织管理、流程、数据、激励、文化、领导力分类。
- 按“问题化拆解”模板生成学习文章：
  - 难度等级
  - 问题类型
  - 问题实质
  - 公司实际解法
  - 原文依据
  - 底层逻辑
  - 案例对照
  - 更好的解法和选择
  - 学习追问
- 学习追问可以点击进入“深挖学习”页面，并继续生成下一层追问。
- 输出结构化梳理、金句清单和六种演化式深化路径。

- 增加多篇讲话的思想演变时间线。
